# Agentrace 重设计 — 讨论记录 & 实施计划

## 最终决策汇总

| # | 决策点 | 结果 |
|---|--------|------|
| 1 | 项目名称 | **Agentrace** |
| 2 | 核心目标 | 全栈分析平台：交互行为 + Safety + 知识域演化 + 自我提升 |
| 3 | 语言 | 全 Rust（CLI + 分析引擎 + Web Server） |
| 4 | 3D 可视化 | Deck.gl + WebGL |
| 5 | 前端 | React + TypeScript + Vite，打包嵌入 Rust binary |
| 6 | UI 组件库 | shadcn/ui + Tailwind CSS |
| 7 | Rust Web 框架 | axum + tokio |
| 8 | Embedding | candle + all-MiniLM-L6-v2 (384-dim BERT, pure Rust) |
| 9 | 存储 | SQLite + sqlite-vec 扩展 |
| 10 | 洞察形态 | Dashboard + LLM 教练分析 |
| 11 | 3D 交互 | 全部 7 项：导航/详情/着色/时间轴/语义搜索/地形图/轨迹线 |
| 12 | 触发方式 | CLI + Daemon 两者 |
| 13 | 交叉分析 | GROUP BY agent × model × task |
| 14 | Workspace | 多 crate 结构 |
| 15 | 构建 | 前端嵌入 Rust binary，单一可执行文件 |
| 16 | LLM 分析 | DeepSeek API (OpenAI-compatible)，conversation 级 coaching |

## Workspace 结构

```
agentrace/
├── Cargo.toml                  # workspace root
├── crates/
│   ├── agentrace-core/         # 数据模型 (Utterance, AgentKind, TaskKind)
│   ├── agentrace-discovery/    # Agent 存储发现 (glob 扫描)
│   ├── agentrace-parser/       # 多格式解析 (JSON/JSONL/MD/text/SQLite)
│   ├── agentrace-storage/      # SQLite 持久化 (utterances, embeddings, graph, coaching)
│   ├── agentrace-embedding/    # BERT embedding (candle, all-MiniLM-L6-v2)
│   ├── agentrace-llm/          # DeepSeek API 客户端 (coaching analysis)
│   ├── agentrace-analysis/     # 分析引擎 (stats, PCA graph, coaching engine)
│   ├── agentrace-cli/          # CLI 入口 (clap derive, 6 commands)
│   └── agentrace-server/       # axum Web 服务器 + 内嵌前端
├── frontend/                   # React + Vite + Deck.gl
├── examples/                   # 测试数据
└── records/                    # 设计讨论记录
```
